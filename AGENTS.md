# Project
- This project is an MNIST digit classifier built entirely only using Python 3 and Numpy, without the use of Machine Learning Libraries like PyTorch or Tensorflow
- THE MNIST digit classifier is built using a Fully Connected Network
- The source code contains classes to corrupt data for testing and experiments
- The weights are initialised using He Normal, and the bias is initially set to 0
